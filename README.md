# PlantUML Black & White Style

Usage:

```plantuml
' diagram.puml

@startuml

!include https://raw.githubusercontent.com/dmitrvk/plantuml-black-and-white/main/base.puml

...

@enduml
```

or:

```
$ wget https://raw.githubusercontent.com/dmitrvk/plantuml-black-and-white/main/base.puml
```

and then:

```plantuml
' diagram.puml

@startuml

!include base.puml

...

@enduml
```
